FJORD VI v51.3 - SOCIO FINAL PREMIUM

Cambios aplicados:
- La acción principal ahora vive arriba, en la tarjeta de Próximo Embarque, y ejecuta la acción real.
- Se redujo la duplicación funcional: el bloque "Tu lugar" queda informativo.
- Se ocultaron bloques repetidos de estado/ocupación que ya estaban expresados arriba.
- Sistema de estados más sobrio: cancelado suave, sin reserva neutro, activo verde.
- Botones secundarios de invitados pasaron a estilo outline para no competir con el CTA principal.
- Se reemplazó la nota permanente de invitados por un botón bloqueado con aviso contextual.
- Bottom nav más liviana: estado activo con indicador sutil, no bloque pesado.
- Sombras más amplias y suaves; menos bordes visibles.

No se modificó:
- Backend.
- Rutas.
- Base de datos.
- Lógica de reservas/cancelaciones/cargos.
- Tooltips.
- Toasts existentes.
- Reglamento.
