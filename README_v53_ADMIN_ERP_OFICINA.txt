FJORD VI v53 ADMIN ERP OFICINA

Rediseño operativo de Administración con criterio de software empresarial:
- Header compacto con versión y usuario.
- Navegación horizontal tipo ERP, sin sidebar ancho.
- Grillas densas y full width.
- Columnas con ancho racional por tipo de dato.
- Nombre/salida fijos para mantener contexto.
- Scroll horizontal superior e inferior sincronizado.
- Filtros compactos y exportar vista filtrada.
- Panel lateral para alta de usuarios.
- Panel lateral para alta de salidas con calendario mensual grande.
- Módulo inicial de estadísticas.

No se modifican Socio, Capitán, Reglamento ni la lógica de reservas/cargos.
